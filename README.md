# 캘린터 투두

<hr/>

# 기획 의도
- 리액트 기본기에 대한 감을 되찾기 위해, 기존에 만든 투두 리스트 클론 코딩에 +a 기능인 캘린더 기능을 더했습니다.
- 일반적으로 투두는 날짜 개념과 묶어서 사용되는 일이 많기에 캘린더 기능을 넣게 되었습니다.

<hr/>

# 사용기술
`SCSS`, `JavaScript: jsx, React.js`, `date-picker`
